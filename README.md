# PopX Authentication App

ReactJS authentication demo using localStorage. Mobile-first responsive design.

## 🚀 Live Demo

**[PopX Authentication App Live](https://popxauthenticationreactjs-oat0f1egl-dsawithsatyas-projects.vercel.app/)**

Deployed on Vercel. Test the app live with full authentication, registration, and account management.

## 🛠️ Local Development

Install and run:

```bash
npm install
npm start
```

Deploy to Vercel:

1. Create a Vercel account at https://vercel.com.
2. Install the Vercel CLI (`npm i -g vercel`) or use the dashboard.
3. From project root run:

```bash
vercel
```

Choose the suggested defaults. Vercel will detect the React app and deploy the `build` output. Alternatively, connect your GitHub repo to Vercel for continuous deployment.

Notes:

- The app uses `localStorage` for persistent users and login state.
- For pixel-perfect mobile layout, the app uses a centered card with max width 400px.
